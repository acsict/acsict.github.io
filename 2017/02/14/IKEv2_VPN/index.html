IKEv2 VPN设置手册

Windows

系统版本要求

Win 7或者更高版本

证书导入

Windows使用IKEv2 VPN需要导入服务器CA证书到信任根证书颁发机构，以Win10为例，Win7、Win8过程类似

  • 下载CA证书到本机，下载地址
  • 开始菜单搜索mmc（Microsoft 管理控制台）
图1-1
  • 文件-添加/删除管理单元，添加证书单元
图1-2
图1-3
  • 证书单元的弹出窗口中选计算机账户，然后选择下一步
图1-4
  • 选择本地计算机/运行此控制台的计算机，然后选择确定
图1-5
  • 在左边的控制台根节点下选择证书 -> 受信任的根证书颁发机构 -> 证书
  • 右键点击证书 -> 所有任务 -> 导入，打开证书导入窗口。
图1-6
图1-7
  • 选择证书文件导入即可

  • 切换到安全选项卡，参数调整为如下，然后确定进行保存
